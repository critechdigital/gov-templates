# CriTech Government Website Templates

Production-ready website templates designed for Philippine Local Government Units (LGUs).

## Templates

| Template | Description | Demo LGU |
|----------|-------------|----------|
| **Transparency Portal** | Budget, procurement, projects, FOI, officials | San Rafael, Bulacan |
| **Permit & Licensing** | Business/building permits, tracking, fee calculators | Meycauayan, Bulacan |
| **Citizen Services** | Service directory, appointments, complaints, emergency | Marilao, Bulacan |
| **Legislative Tracker** | Ordinances, resolutions, sessions, committees | San Jose del Monte, Bulacan |
| **Procurement Portal** | Bidding, contracts, BAC, procurement plan (RA 9184) | Provincial Gov of Bulacan |

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS
- React Router
- Recharts
- Lucide Icons

## Build

```powershell
powershell -File build.ps1
```

Outputs all templates to `dist/` with a landing page.

## Deploy

Connected to Vercel. Push to `main` to deploy.
